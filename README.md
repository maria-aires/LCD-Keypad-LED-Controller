# LCD Keypad LED Controller

## 📖 About

This project was developed for the **Introduction to Robotics** course at the Federal University of Catalão (UFCat).

The system uses an Arduino Uno together with a 16x2 LCD Keypad Shield to control two LEDs through an interactive menu displayed on the LCD.

Each button performs a different action:

- Turn the green LED ON
- Turn the red LED ON
- Turn both LEDs OFF
- Blink both LEDs
- Display the selected action on the LCD

---

## 🔧 Hardware

- Arduino Uno
- LCD Keypad Shield 16x2
- Green LED
- Red LED
- Breadboard
- Resistors
- Jumper wires

---

## 💻 Software

- Arduino IDE
- C/C++
- LiquidCrystal Library

---

## ✨ Features

- Interactive LCD menu
- LED control
- Analog button reading
- LCD feedback
- Modular code using functions
